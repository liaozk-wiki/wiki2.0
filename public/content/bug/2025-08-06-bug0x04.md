# 0x04OGNL失效

# 问题描述：

mybatis动态where中条件如下：

```xml
            <if test="dto.allDsr == 0">
                and 1 = 1
            </if>
            <if test="dto.allDsr == '0'">
                and 2 = 2
            </if>
            <if test='dto.allDsr == "0"'>
                and 3 = 3
            </if>
```







参数dto中allDsr是String类型的"0"。why 最终结果中有1=1，3=3 却没有2=2？
